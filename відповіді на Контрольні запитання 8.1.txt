                 Контрольні запитання
1. Як створити новий репозиторій? :: Залогінитись, та зайти на головну сторінку GitHub. Натиснути на кнопку створення нового репозиторію. Обрaти ім'я для репозиторію (Repository name). При створенні нового репозиторію обрати “Initialize this repository with a README”.
2. Для чого призначен репозиторій? :: для відстеження git
3. Як створити нову гілку у репозиторії? :: Для створення гілки потрібно натиснути на кнопку Branch master та у списку, що випадає, ввести ім’я нової гілки
4. Як злити гілки з основною гілкою master? :: Для цього натиснути на кнопку New pull request. На сторінці запиту на злиття буде спочатку перевірено можливість злиття
гілок. Після перевірки та можливості злиття натисніть на кнопку Create pull request
5. Як завантажити репозиторій до себе на комп’ютер? :: Для завантаження репозиторію на комп’ютер необхідно натиснути на кнопку Clone or Download, потім Download ZIP
